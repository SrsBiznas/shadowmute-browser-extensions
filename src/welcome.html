Shadowmute.
Thanks for installing the Shadowmute Browser extension Please note, to use this extension, a functioning Shadowmute account is required. To register for a Shadowmute account, please visit shadowmute.com.

With this extension you can:
  • Create a mailbox
  • Log out
Limitations To protect your privacy, the Shadowmute browser extension only has limited functionality to create new mailboxes.
Viewing messages for your mailboxes requires you to log in to the main site. Support If you have a problem with this extension, please email Support. Alternatively, contacting via twitter may be faster.
